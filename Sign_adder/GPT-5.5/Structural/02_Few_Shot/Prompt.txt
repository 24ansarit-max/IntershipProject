2. Few-Shot Prompt  
Prompt- 
I will show you examples of structural Verilog modeling. You will extend them to a full 
16-bit signed adder with testbench. 
 
STRICT RULE: The top-level adder module must be built ONLY by instantiating 
sub-modules 
and wiring them together. No assign, no always, no arithmetic operators inside the 
top-level structural module. Logic only lives inside primitive sub-modules. 
 
EXAMPLE 1 — 1-bit half adder (primitive level): 
module half_adder(input a, b, output sum, cout); 
  xor g1(sum, a, b); 
  and g2(cout, a, b); 
endmodule 
 
EXAMPLE 2 — 1-bit full adder using structural instantiation: 
module full_adder(input a, b, cin, output sum, cout); 
  wire s1, c1, c2; 
  half_adder ha1(.a(a), .b(b), .sum(s1), .cout(c1)); 
  half_adder ha2(.a(s1), .b(cin), .sum(sum), .cout(c2)); 
  or  g1(cout, c1, c2); 
endmodule 
 
EXAMPLE 3 — 4-bit structural ripple carry adder: 
module adder4_struct(input [3:0] A, B, input cin, output [3:0] sum, output cout); 
  wire [4:0] c; 
  assign c[0] = cin; 
  full_adder fa0(.a(A[0]),.b(B[0]),.cin(c[0]),.sum(sum[0]),.cout(c[1])); 
  full_adder fa1(.a(A[1]),.b(B[1]),.cin(c[1]),.sum(sum[1]),.cout(c[2])); 
  full_adder fa2(.a(A[2]),.b(B[2]),.cin(c[2]),.sum(sum[2]),.cout(c[3])); 
  full_adder fa3(.a(A[3]),.b(B[3]),.cin(c[3]),.sum(sum[3]),.cout(c[4])); 
  assign cout = c[4]; 
endmodule 
 
NOW DO THE FOLLOWING: 
1. Use EXAMPLE 1 as your half_adder primitive. 
2. Use EXAMPLE 2 as your full_adder built from half_adder instances. 
3. Scale EXAMPLE 3 to a 16-bit signed adder (adder16_struct): 
   - Instantiate exactly 16 full_adder modules (fa0 through fa15) 
   - Wire carry chain: c[0] through c[16] 
   - c[0] = 0 (no initial carry for addition) 
   - Output: wire the 16 sum bits + c[16] as the 17th bit (overflow) 
   - Declare all ports as signed 
 
4. Build a testbench with ALL four loop types: 
   - for loop: 20 boundary corner case pairs 
   - while loop: 100 iterations with $signed($random) inputs 
   - repeat loop: 50 iterations with random inputs 
   - forever loop: clock toggle every 5ns, $finish at 1000ns 
   Include PASS/FAIL checking and $display in every loop.
