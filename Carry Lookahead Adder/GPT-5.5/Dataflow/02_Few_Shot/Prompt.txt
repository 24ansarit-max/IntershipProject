EXAMPLE — Ripple Carry Adder (slow, simple): 
```verilog 
module rca_16bit(input [15:0] a,b, input cin, output [15:0] sum, output cout); 
  wire [16:0] c; 
  assign c[0]=cin; 
  genvar i; 
  generate for(i=0;i<16;i=i+1) 
    assign {c[i+1],sum[i]} = a[i]+b[i]+c[i]; 
  endgenerate 
  assign cout=c[16]; 
endmodule 
// Delay: O(N) — 16 gate levels 
```

2. Intra-group carry assigns (groups of 4) 
3. Group-level G/P (GG, GP) 
4. Inter-group carries (C4, C8, C12) 
5. Sum assigns
